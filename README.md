# AWSGITHUBACTIONS
This repository contains a  Github actions workflow to deploy Hello World App to S3 
Pre-Requisites:
  You need Access and secret access keys for your AWS account.
  Go to settings tab of your repository and scrol down. On left handside expand "Secrets and Variables". 
  Click on actions. create Access and secret access keys here.

Create a bucket. Enable public access and static website hosting.

Run your workflow and see the deployment.
